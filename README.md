# PySpectra

Python interface to the C++ Spectra library

## Installation
To install pyspectra, do:
```bash
  git clone https://github.com//pyspectra.git
  cd pyspectra
  pip install .
```

Run tests (including coverage) with:

```bash
  python setup.py test
```
